# Bookmark-go
个人导航

> 属于小白练手，如果有什么可以优化的地方请大佬们赐教 🙏
> 
> 本应用后端用gin进行开发。前端偷懒直接用了基础的H5而已，看官们可以自行添加在skin目录下

### 构建使用
```shell
go build -o ./bin/run.serv main.go
./bin/run.serv
```
